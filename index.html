We Should Go The Portal Route

A proper Xenforo portal addon could convert threads in select nodes into articles to display on a front page. The OP would be the article and the replies would show up as comments. This is not to the exclusion of normal forum usage, but it would create a more modern first impression. An off-site portal such as this is possible. We can host a front page that displays dynamic content with little to no maintenance required thanks to Xenforo's RSS feeds. I want to stress that this is not ideal, but it is doable.
